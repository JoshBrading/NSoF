# Not Sea of Friends (NSoF)
## About
NSoF is a Sea of Thieves fan game made in Unity. 
## Controls
| Key  | Command |
| :---:  | :--- |
| W, A, S, D | Movement  |
| Left Mouse  | Attack  |
| Shift  | Sprint  |
| F  | Enter/Pickup  |
| X  | Exit/Drop  |
| P  | Force quit quest  |
| Esc + L Mouse  | Exit  |
